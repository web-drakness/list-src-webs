🔐 Authentication

ส่ง API Key ผ่าน Authorization header หรือ body parameter

Header (แนะนำ)
Authorization: Bearer YOUR_KEY
Body
api_key=YOUR_KEY
API Key มีความยาว 32 ตัวอักษร สร้างได้จาก Dashboard สามารถล็อค IP เพิ่มความปลอดภัย
🧧

TrueWallet ซองอั่งเปา

รับซองของขวัญ TrueWallet Gift Voucher อัตโนมัติ

vv1
Endpoint
POST /api/v1/truewallet/gift
Parameters
FieldTypeRequiredคำอธิบาย
voucher_linkstringrequiredลิงก์ซองอั่งเปา TrueWallet
mobilestringoptionalเบอร์กระเป๋าปลายทาง
Response
{
  "status": "success",
  "message": "รับซองสำเร็จ",
  "data": {
    "amount": 50,
    "voucher": "XXXXXXXXXXXXXX"
  }
}
ตัวอย่างโค้ด
$ch = curl_init("./index.html/api/v1/truewallet/gift");
curl_setopt_array($ch, [
  CURLOPT_POST => true,
  CURLOPT_RETURNTRANSFER => true,
  CURLOPT_HTTPHEADER => [
    "Authorization: Bearer YOUR_KEY",
    "Content-Type: application/json"
  ],
  CURLOPT_POSTFIELDS => json_encode([
    "voucher_link" => "https://gift.truemoney.com/campaign/?v=XXXXX"
  ])
]);
$result = json_decode(curl_exec($ch), true);
echo $result['message'];
const response = await fetch("./index.html/api/v1/truewallet/gift", {
  method: "POST",
  headers: {
    "Authorization": "Bearer YOUR_KEY",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    voucher_link: "https://gift.truemoney.com/campaign/?v=XXXXX"
  })
});
const data = await response.json();
console.log(data.message);
import requests

response = requests.post(
  "./index.html/api/v1/truewallet/gift",
  headers={"Authorization": "Bearer YOUR_KEY"},
  json={"voucher_link": "https://gift.truemoney.com/campaign/?v=XXXXX"}
)
print(response.json()["message"])
const axios = require("axios");

const response = await axios.post(
  "./index.html/api/v1/truewallet/gift",
  { voucher_link: "https://gift.truemoney.com/campaign/?v=XXXXX" },
  { headers: { Authorization: "Bearer YOUR_KEY" } }
);
console.log(response.data);

⚠️ Error Codes

HTTPStatusความหมาย
200 success ดำเนินการสำเร็จ
400 fail ข้อมูลไม่ถูกต้อง หรือ voucher ไม่ถูกต้อง
401 fail API Key ไม่ถูกต้อง หรือถูกปิดใช้งาน
402 fail ไม่มีแพ็กเกจที่ใช้งานได้ กรุณาซื้อแพ็กเกจ
403 fail บัญชีถูกระงับ หรือ IP ไม่ได้รับอนุญาต
422 fail ข้อมูลไม่ครบถ้วน
503 error ระบบปิดปรับปรุงชั่วคราว
